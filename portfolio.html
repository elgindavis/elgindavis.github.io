My Daily Bread

Designed and Created "My Daily Bread" for Android using Ionic and Google Firebase

Click here to learn more

My Daily Bread serves to allow believers to track the verses that they read every day, as well as keep them accountable for the days that they miss. Even further, you can now keep track of your thoughts, ideas, and spiritual experiences that supplement each passage you read.

I created My Daily Bread over the course of 3 weeks through the Ionic Framework, using HTML, SCSS, and Typescript. The app's database is linked to Firebase, so each user's saved verses and reflections are saved in Google's databases, rather than being stored locally. Thus, this application requires a stable internet connection to function.

Currently available in the Google Play store!

SUBSPACE

SubSpace was created as a final project in CS50 by myself and Jack Kelley. The purpose of SubSpace is to allow students to access the college's official events calendar, while also having the ability to submit events to be added to an unofficial student calendar. It makes use of a Google-powered campus map to locate the events. SubSpace will help you to stay on top of all of the Harvard-sponsored events, while also creating and sharing "low-key" events within the student body.

The application was created using HTML and CSS for the interface, and PHP and SQL to handle the subscriptions and sign-ins, as well as the user-submitted events. The calendar feature in the application was powered by Google Calendar, and the entire application was hosted within CS50's ide50 IDE.
